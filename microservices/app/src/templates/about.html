SHARK@JNU

Shark@JNU is a demo application which provides a platform for customers to find the number of sims linked to the account number.Department of Telecommunication can also use this to search a particular aadhar number and can see the number of mobile numbers attached to that particular aadhar number. Telecom service providers will use the application to add entry to database of particular mobile number which is going to be linked. Our support team will do the registration of Telecom service providers and Department of Telecommunication professional for security purposes.
